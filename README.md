# ChatGPTBot
GPT-3 backed Discord bot to chat with.
### Getting started:
```
docker build -t chatbot . && docker run --env DISCORD_TOKEN='abc123' --env GPT_TOKEN='abc123' chatbot
```
### Bot Commands:
```
/chat [prompt] - Chat with the bot.
```
